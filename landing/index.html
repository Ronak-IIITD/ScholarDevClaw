Frequently asked questions

ScholarDevClaw is an open source AI coding agent that analyzes your codebase, finds relevant research improvements from the latest ML papers, and generates validated patches automatically. It supports Python, JavaScript, TypeScript, Go, Rust, and Java.

Install it with the one-line command above, then run scholardevclaw --help to see all available commands. Use scholardevclaw demo for a quick demo, or scholardevclaw tui for the interactive terminal interface.

ScholarDevClaw works with a built-in knowledge base of 16+ paper specs. For advanced features like LLM-powered research and semantic matching, you can connect any model from any provider (Claude, GPT, Gemini, etc.).

ScholarDevClaw runs entirely on your local machine. Your code never leaves your computer unless you explicitly configure external services. All analysis and patch generation happens locally.

Yes! ScholarDevClaw is fully open source under the MIT license. You can contribute, fork, or modify it however you like. Check out our GitHub repository for the source code.
