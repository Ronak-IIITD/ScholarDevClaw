Capabilities

Everything you need to modernize
your ML codebase

From repository analysis to validated production patches — ScholarDevClaw automates the entire research-to-code workflow.

Repository Intelligence

AST-level code analysis with tree-sitter. Detects frameworks, architectures, element names, imports, and code patterns automatically.

Research Intelligence

16 built-in paper specs covering normalization, attention, activation, positional encoding, and optimizers. Live arXiv discovery for new papers.

Smart Mapping Engine

6-tier matching: exact, fuzzy, import, text-scan, legacy-architecture, and LLM semantic. Finds targets with real file:line locations.

CST Patch Generation

Safe AST-level code transformations via libcst. 10+ built-in transformers. Safe rollback on failure. No manual diff editing.

Benchmark Validation

Runs real micro-benchmarks with PyTorch or pure Python fallback. Measures speedup, memory, loss delta. Only merges validated patches.

Interactive TUI + API

Rich Textual TUI with wizard workflow, agent REPL, and live streaming. FastAPI server for programmatic integration. CLI for scripts.

Workflow

From idea to validated patch
in minutes

A fully automated pipeline that turns ML research into production-ready code changes.

1

Analyze your repository

Tree-sitter AST analysis detects your ML framework, model architecture, layers, and imports. scholardevclaw analyze ./repo

2

Discover research improvements

Match against 16 built-in paper specs. Optional live arXiv search with LLM extraction discovers cutting-edge techniques.

3

Map specs to code locations

6-tier mapping engine finds exact insertion points with real file:line targets. Validates compatibility with your codebase.

4

Generate safe patches

libcst AST transformers apply changes safely. Automatic rollback on failure. New files from proven templates.

5

Validate with benchmarks

Real micro-benchmarks measure speedup and memory. Scorecard shows pass/fail per check. Only validated patches proceed.

6

Integrate to production

Apply validated patches to your repo. Git-backed rollback. Hook system for CI integration. Full audit trail.

Get Started

One command to install

Works on macOS, Linux, and WSL2. Requires Python 3.10+.

Shell (Recommended)

The installer script handles everything — Python check, dependencies, and CLI setup.

$ curl -fsSL https://Ronak-IIITD.github.io/ScholarDevClaw/install.sh | bash

pip / pipx

Install via Python package manager. Requires Python 3.10 or later.

$ pip install scholardevclaw

pipx (Isolated)

Run in an isolated environment. Best for keeping your system Python clean.

$ pipx install scholardevclaw

From Source

Clone and install manually. For contributors and advanced users.

$ git clone https://github.com/Ronak-IIITD/ScholarDevClaw && cd ScholarDevClaw/core && pip install -e .
